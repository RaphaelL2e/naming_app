# 取名小程序 (Naming App)

> 微信小程序 - 为宝宝取好名字

## 项目结构

```
naming-app/
├── app.js              # 小程序入口
├── app.json           # 全局配置
├── pages/
│   └── index/         # 首页
│       ├── index.wxml  # 页面结构
│       ├── index.wxss  # 页面样式
│       └── index.js   # 页面逻辑
├── utils/             # 工具函数
└── images/            # 图片资源
```

## 功能

1. **输入姓氏** - 用户输入姓氏
2. **一键生成** - 点击按钮生成5个好名字
3. **名字含义** - 显示每个名字的含义

## 运行

在微信开发者工具中打开 `naming-app` 目录即可预览。
